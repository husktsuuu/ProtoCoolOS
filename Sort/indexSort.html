Guía de Uso – Módulo SORT

1. Manejo de Listas

• Agregar Lista: permite ingresar una lista manualmente.
• Generar Random: genera una lista aleatoria.
• Limpiar: borra todo.
• Guardar: exporta la lista actual.
• Importar: carga una lista guardada.

2. Algoritmos de Ordenamiento

Seleccione el algoritmo que desea aplicar:
• Selection Sort
• Insertion Sort
• Shell Sort
• Merge Sort

Cada uno tiene dos modos:
• Ordenar de Menor a Mayor
• Ordenar de Mayor a Menor

3. Visualización

• El gráfico de barras muestra visualmente la lista.
• Los cambios se actualizan automáticamente después de ordenar.
• Se muestra también:
- Lista original
- Lista ordenada
- Tiempo total de ejecución

4. Consejos

• Se recomienda generar listas cortas para ver claramente el efecto del algoritmo.
• Los algoritmos tienen diferente velocidad, útil para compararlos.
• Use guardar/importar para probar configuraciones repetidas.
