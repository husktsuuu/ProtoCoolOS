AYUDA - ALGORITMO DE KRUSKAL


CREACIÓN DEL GRAFO

- Clic en un espacio vacío: crea un nuevo nodo.

- Clic sobre un nodo y luego sobre otro: crea una arista entre ellos.

- Al crear una arista, el sistema solicitará ingresar un peso numérico.


EDICIÓN

- Clic derecho sobre un nodo: renombra el nodo.

- Clic derecho sobre una arista: modifica su peso.

- Botón “Eliminar”: permite eliminar nodos o aristas con un clic.


COLORES

- Los selectores permiten cambiar el color de nodos, texto y aristas nuevas.

- Botón “Borrar Colores”: restaura los colores originales del grafo.


KRUSKAL

- Botón “Árbol Mínimo”: aplica el algoritmo de Kruskal para obtener un árbol de expansión mínima.

- Botón “Árbol Máximo”: aplica la versión modificada para generar un árbol de expansión máxima.

- El resultado muestra la suma de los pesos seleccionados.


EXPORTAR E IMPORTAR

- Botón “Exportar”: permite guardar el grafo como PNG, PDF o JSON editable.

- Botón “Importar”: carga un archivo JSON previamente exportado.

- Botón “Limpiar”: reinicia completamente el grafo.
