¿Qué son los algoritmos?

Un algoritmo es un conjunto ordenado de pasos definidos y finitos que se utilizan para resolver un problema o realizar una tarea específica. Son la base de la programación y pueden ser simples o complejos, pero todos siguen reglas para producir un resultado deseado.
