Guía de Uso - Método de Esquina Noroeste

1. Creación de nodos

• Haga clic en el área del grafo para crear un nodo.
• El sistema solicitará un nombre (por defecto Nodo 1, Nodo 2, etc.).

2. Creación de aristas

• Seleccione un nodo origen y luego un nodo destino.
• Ingrese el costo o peso correspondiente.

3. Edición de elementos

• Clic derecho en un nodo o arista para cambiar su nombre o valor.

4. Colores

• El selector de color permite definir el color de los nuevos nodos.

5. Eliminación

• Active el modo eliminar y haga clic sobre un nodo o una arista.
• Vuelva a presionar “Eliminar” para salir del modo.

6. Matriz de costos

• La matriz se actualiza automáticamente con las aristas creadas.
• Representa las rutas, las ofertas y las demandas.
• Puede editar manualmente las casillas de costo, oferta y demanda.

7. Método Esquina Noroeste

• Minimizar: aplica el método NW clásico para obtener el costo mínimo.
• Maximizar: aplica la variación para obtener el beneficio máximo.

8. Balanceo automático

• Si la oferta y demanda no coinciden, se agrega automáticamente una fila o columna ficticia.

9. Exportación

• Exportar PNG: genera imagen del grafo.
• Exportar PDF: genera esquema del grafo.
• Exportar JSON: guarda nodos, aristas y estado editable.

10. Importación

• Puede cargar un archivo JSON para restaurar un grafo previo.

11. Limpieza

• Elimina todos los nodos, aristas, matriz y resultados.
• Restablece todo el módulo a su estado inicial.
